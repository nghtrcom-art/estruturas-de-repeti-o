# Atividade sobre estruturas de repetição

**Atividade proposta para a prática de estruturas de repetição em Java**

**O programa irá pedir ao usuário:**

- **O nome**

- **Um número**

**Com esses dados fornecidos o programa irá:**

- **Verificar se a quantidade de letras é superior a 6, se sim 
exibirá o nome na tela pela mesma quantidade de letras, caso não
exibirá o nome do usuário apenas uma vez**


- **Ler o número digitado e realizar a contagem de 0 até a quantidade
do número digitado, e contar tanto em ordem crescente quanto decrecente.**
